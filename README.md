# Asteroids.py (Pygame)

A simple Asteroids game implementation using Pygame, created as part of the Boot.dev track.

## Quick Start

1. Ensure Python and Pygame are installed
2. Run: `python main.py`

## Controls

- Arrow keys: Move
- Spacebar: Shoot
